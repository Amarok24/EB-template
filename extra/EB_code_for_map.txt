1) import MAPBOX-GL JS and CSS code (save offline)

<script src='https://api.tiles.mapbox.com/mapbox-gl-js/v0.46.0/mapbox-gl.js'></script>
<link href='https://api.tiles.mapbox.com/mapbox-gl-js/v0.46.0/mapbox-gl.css' rel='stylesheet' />


2) import EB-template-map.fnp.js (and edit code at the beginning)
    
<script src="scripts/EB-template-map.fnp.js">/* EB MapboxMap */</script>


3) place the map into HTML code

    <div id="myMapboxMap">
        <!-- leave this empty -->
    </div>

4) style your map as you wish, for example:

<style>
	#myMapboxMap {
		width: 500px;
		height: 600px;
		margin: 3em auto 1em;
		border: 1px solid #dfdfdf;
	}

		background-image: url(marker-icon.png);
		background-size: cover;
		width: 25px;
		height: 41px;
		cursor: pointer;
	}

	#myMapboxMap .marker:hover {
		background-image: url(marker-icon-hover.png);
	}

	#myMapboxMap .mapboxgl-popup {
		max-width: 250px;
	}

	#myMapboxMap .mapboxgl-popup-content {
		text-align: center;
		padding-top: 1.5em;
	}

	#myMapboxMap .mapboxgl-popup-content big {
		display: block;
		font-size: 1.4em;
		font-weight: bold;
		margin-bottom: .5em;
	}

	#myMapboxMap .mapboxgl-popup-content p {
		margin-bottom: 0;
	}

	#myMapboxMap .mapboxgl-popup-content a {
		color: cornflowerblue;

	}

	}

	#myMapboxMap .mapboxgl-canvas {
		position: relative !important; /* bugfix, default "absolute" is a mess in hidden DIVs even after resize(); */
	}
